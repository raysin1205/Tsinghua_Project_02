## 封路场景

**封路位置**：R15 学堂路从北端听涛园 / 丁香园一带，一路向南切断到经管楼附近，共 **5 段连续封闭、10 条有向边**。

| 序号 | 封闭路段 | 边 ID | 长度 |
|---|---|---|---|
| ① | **听涛园（J06） ↔ 丁香园（T06）** | E025 | 169.5 m |
| ② | **清芬园（J12） ↔ 听涛园（J06）** | E037 | 205.6 m |
| ③ | **第四教室楼（C01） ↔ 清芬园（J12）** | E048 | 147.4 m |
| ④ | **第五教室楼（J24） ↔ 第四教室楼（C01）** | E066 | 122.0 m |
| ⑤ | **经管新楼（C04） ↔ 第五教室楼（J24）** | E079 | 138.2 m |

**总封闭长度约 783 m**。

这条南北纵线穿过 **丁香园食堂 → 听涛园 → 清芬园食堂 → 第四教室楼 → 第五教室楼 → 经管楼**，是清芬园食堂前后约 800 米的整条主路。

代码里的封路声明：
```python
blocked = block_edge_pairs（net, [
    （"J06", "T06"）,  # 听涛园 ↔ 丁香园
    （"J12", "J06"）,  # 清芬园 ↔ 听涛园
    （"C01", "J12"）,  # 第四教室楼 ↔ 清芬园
    （"J24", "C01"）,  # 第五教室楼 ↔ 第四教室楼
    （"C04", "J24"）,  # 经管楼 ↔ 第五教室楼
]）
```

## 注意

- 节点的 `name` 字段是地点名，**不要用 `name` 做索引**，所有逻辑都用 `node_id`（T01、D01、J01 等）。
- 边表里 `bidirectional=true` 表示双向通行，在 `Network` 类里会被拆成 `E001f`（正向）和 `E001b`（反向）两条有向边。
- 封路一次封掉一对无向边的**两个方向**，所以封 5 段 = 10 条有向边。
- 释放曲线 `release_curve.csv` 是下课后 8 分钟内的释放强度，峰值在 2 分钟左右。
